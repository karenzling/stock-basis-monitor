## 一、核心方法论：紫苏叶理论

顶级寿司店里，食客追逐金枪鱼大腹，但真正不能断供的是伊豆小农场的紫苏叶。没有金枪鱼，菜单减几道菜；没有紫苏叶，整家店停摆。

**Serenity 的投资逻辑 = 找到 AI 产业链里的紫苏叶。**

1. **从上往下拆**：跳过市场热炒的 GPU/大模型（金枪鱼），追问"当 AI 集群规模扩大时，哪个环节最先撑不住？"
2. **找物理瓶颈**：铜线互连在功耗和带宽上有物理极限 → CPO（光电共封装）是绕不过的路径
3. **卡脖子节点筛选**：CPO 路径上找出 5 个最窄的节点
4. **选标的标准**：小市值、无机构覆盖、处于物理瓶颈处、等待机构轮动验证
5. **研究验证**：啃材料学论文 + 分析专利 + 追踪产能扩张 + 供应链认证周期 + 出口管制动态 → 输入 AI 模型做对抗性压力测试（找漏洞、替代方案威胁、估值偏差）→ 通过才公开发布

## 二、筛选标准 #1：融资结构与流通盘动态

**这是 Serenity 筛选标的的首要标准。** 一家公司业务再好，如果融资结构有毒，股东价值会被持续稀释。

### 2.1 融资结构类型判断

- **Toxic ATM** — 无限增发，每次上涨都卖。例：$IREN（$6B ATM）。红牌，远离
- **最优结构** — 战略直投 + 可转债组合。例：$NBIS（NVDA直投，YTD+153%）。绿牌
- **高息债务** — 高利率贷款，利息吃掉FCF。例：$CRWV（高息贷款买GPU）。长期有毒
- **迷你型ATM** — 市值远小于ATM额度。例：$SLNH（$250M市值/$500M ATM）。送钱给公司
- **SBC掩护** — 股权激励包装成盈利。例：$SNAP。稀释股东
- **管理层薪酬泵** — 无限增发养高管。例：$BKKT。红牌

### 2.2 如何评估

1. **查 ATM 规模**：对比市值 vs ATM 剩余额度
2. **查债务结构**：利率、到期时间、是否可变利率
3. **查股权激励（SBC）**：占营收比例、是否被算入Non-GAAP盈利
4. **查融资历史**：是否每次股价涨就有增发
5. **查管理层持股**：是否管理层也在卖

### 2.3 Serenity 的策略

- **好公司 + 有毒结构**：等现有持有人被稀释干净后再考虑做多
- **好公司 + 好结构**：积极进入
- **公司一般 + 任何结构**：跳过

## 四、核心持仓

### 1. $AXTI — AXT Inc（InP 衬底）

- **定位**：全球 InP 衬底产能占比 25%-35%，霍尔木兹海峡级别的卡位
- **逻辑**：AI 光通信依赖 InP 衬底，没有它底层材料链断掉
- **买入时**：市值约 $2 亿，几乎无机构覆盖
- **路径**：$12 → $70+（最高浮盈 1000%+），目标价 $150
- **状态**：已大幅兑现利润

### 2. $SIVE — Sivers Semiconductors（CPO 激光光源）

- **定位**：CPO 架构外置连续波激光光源，最稀缺上游资产之一
- **买入时**：市值约 $1.5 亿
- **涨幅**：20x+
- **催化剂**：$AVGO 或 $MRVL 可能以 $2-3 亿收购，锁定 CPO 激光供应链
- **状态**：当前最坚定持仓

### 3. $AAOI — Applied Optoelectronics（光收发器）

- **定位**：光收发器全链条玩家（激光器设计 → 组装 → 销售）
- **买入时**：市值约 $6.6 亿，认为被严重低估
- **涨幅**：7x+
- **逻辑**：AI 光模块需求爆发
- **状态**：你的监控列表已有

### 4. $RPI — Raspberry Pi（边缘计算基座）

- **定位**：大多数认为是廉价教育单板电脑 → 实际是 AI 初创公司的物理隔离部署基座
- **逻辑**：AI 创业公司大量采购 RPI 做群控系统部署基座，需求被市场严重低估
- **战绩**：提前 2 个月预测全年营收增速 58%（市场一致预期 14%），财报后单日 +44%，次日 +27%

### 5. $XFAB — X-FAB（SiC 代工 + 硅光子）**最新建仓**

- **定位**：欧美唯一高量产 SiC 代工厂，同时具备硅光子代工能力
- **市值**：约 $12.8 亿
- **市净率**：1.29
- **催化剂**：EU CHIPS Act 2 预计近期落地；英伟达和诺基亚硅光子评估进行中
- **标签**：小市值、无覆盖、物理瓶颈、等待机构轮动验证 — 和所有核心标的高度一致

## 六、整体画像

- **选股风格**：小市值（$1-15 亿）、无机构覆盖、冷门赛道
- **底层主线**：AI 数据中心光互连 + CPO 架构
- **研究方法**：材料学论文 + 专利分析 + 供应链追踪 + 出口管制
- **验证方法**：AI 对抗性压力测试（找漏洞/替代威胁/估值偏差）
- **首要筛选**：融资结构与流通盘动态（ATM/债务/SBC）
- **发布形式**：X 长线程（深度研究帖）
- **仓位特征**：25 只公开建仓，涨幅 1x-10x
- **最大风险**：高度集中（底层逻辑一致），若 CPO 路线变或推进慢，组合同时受影响
- **流动性风险**：小公司流动性有限，大资金建仓/退出困难
