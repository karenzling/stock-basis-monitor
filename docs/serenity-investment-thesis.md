## 一、核心方法论：紫苏叶理论

顶级寿司店里，食客追逐金枪鱼大腹，但真正不能断供的是伊豆小农场的紫苏叶。没有金枪鱼，菜单减几道菜；没有紫苏叶，整家店停摆。

**Serenity 的投资逻辑 = 找到 AI 产业链里的紫苏叶。**

1. **从上往下拆**：跳过市场热炒的 GPU/大模型（金枪鱼），追问"当 AI 集群规模扩大时，哪个环节最先撑不住？"
2. **找物理瓶颈**：铜线互连在功耗和带宽上有物理极限 → CPO（光电共封装）是绕不过的路径
3. **卡脖子节点筛选**：CPO 路径上找出最窄的 5 个节点
4. **选标的标准**：小市值、无机构覆盖、处于物理瓶颈处、等待机构轮动验证
5. **研究验证**：啃材料学论文 + 分析专利 + 追踪产能扩张 + 供应链认证周期 + 出口管制动态 → 输入 AI 模型做对抗性压力测试（找漏洞、替代方案威胁、估值偏差）→ 通过才公开发布

## 二、核心时间差："扩产周期 vs 需求增速"

### 2.1 核心认知差

- **市场的看法**：稀缺是暂时的，等产能上来就平了
- **Serenity 的看法**：等产能上来，需求又翻倍了，你仍然不够

**利润来源 = 扩产周期（年） × 需求增速（%/年）_ 的时间差。**

### 2.2 判断标准

评估一个标的时，必须回答两个问题：

1. **扩产需要多久？**（从下单到满产，含设备交货、安装调试、良率爬坡）
   - 2年+ → 好（供给锁死，时间差足够大）
   - 1年以内 → 不好（很快就不稀缺了）
   - 6个月以内 → 赌不了，已经有人在扩了

2. **需求增速多快？**（下游客户的采购量增长）
   - 翻倍级（YoY 50%+）→ 好
   - 稳定增长（YoY 10-20%）→ 一般
   - 缓慢增长（YoY <10%）→ 不好

### 2.3 四种组合

| 扩产周期 | 需求增速 | 结果 |
|----------|----------|------|
| 长（2年+） | 快（翻倍级） | ✔️ **最肥的机会**。供给永远追不上，价格持续涨 |
| 长（2年+） | 慢（<10%） | ❌ 产能迟早过剩 |
| 短（<1年） | 快（翻倍级） | ⚠️ 短期脉冲，等别人扩完就没了 |
| 短（<1年） | 慢（<10%） | ❌ 没机会 |

### 2.4 AXTI 验证案例

- 扩产周期：InP 衬底长晶炉订 + 工艺调试 + 良率爬坡 ≈ 18-24 个月
- 需求增速：AI 集群部署导致 800G/1.6T 光模块需求翻倍增长
- 时间差：2022 年看 → 2024-2025 年供需最紧张 → 股价从 $2 亿涨到 $70+
- **结论：完美匹配**

## 三、筛选标准 #1：融资结构与流通盘动态

**这是 Serenity 筛选标的的首要标准。** 一家公司业务再好，如果融资结构有毒，股东价值会被持续稀释。

### 3.1 融资结构类型判断

- **Toxic ATM（市价增发）** — 公司装了个印股票机器，股价一涨就印新股卖钱。例：$IREN（$6B ATM）。红牌，远离
- **最优结构** — 拿了战略投资或发了可转债，不靠卖股吸血。例：$NBIS（NVDA直投，YTD+153%）。绿牌
- **高息债务** — 借钱利率高，赚的钱还利息。例：$CRWV（高息贷款买GPU）。长期有毒
- **迷你型ATM** — 市值$250M却有$500M ATM额度，离谱。例：$SLNH。送钱给公司
- **SBC掩护（股权激励）** — 公司表面盈利，实为发股票当工资藏着。例：$SNAP。稀释股东
- **管理层薪酬泵** — 印股票不为发展，就为养高管。例：$BKKT。红牌

### 3.2 如何评估

1. **查 ATM 规模**：市值 vs ATM 剩余额度
2. **查债务结构**：利率、到期时间、是否可变
3. **查 SBC**：占营收比例、是否藏在Non-GAAP里
4. **查融资历史**：是否每次涨就增发
5. **查管理层持股**：是否管理层也在卖

### 3.3 策略

- 好公司 + 有毒结构 → 等现有持有人被稀释干净后再考虑做多
- 好公司 + 好结构 → 积极进入
- 公司一般 + 任何结构 → 跳过

## 五、完整的股票评分流程

### Step 1：查融资结构（红牌机制）
- 有无巨量 ATM？→ 有则暂时跳过
- 有无高息债务压 FCF？→ 有则扣分
- SBC 占比高且藏起来？→ 扣分

### Step 2：定位物理瓶颈
- 这个公司处于 AI 产业链哪个环节？
- 是否是 CPO / 光互联 / 稀缺材料的必经节点？
- 有没有替代方案？替代方案多久能成熟？

### Step 3：评估时间差（核心利润来源）
- **扩产周期多长？**（设备交期、建厂、认证、良率爬坡）
- **需求增速多快？**（下游客户扩产节奏、资本开支趋势）
- **时间差够不够大？**（2年+ vs 翻倍级 → 最肥）

### Step 4：催化剂判断
- 有没有明确的"扳机"？（大客户认证、政策落地、竞争对手退出、财报验证）
- 催化事件在什么时间窗口？

### Step 5：估值与流动性
- 当前市值、是否被机构覆盖
- 日均交易量能否容纳你的仓位

### 评分维度权重

| 维度 | 权重 | 说明 |
|------|------|------|
| 融资结构 | 红牌否决 | 有毒直接跳过 |
| 物理瓶颈 | 30% | 是否在卡脖子节点上 |
| 扩产周期×需求增速 | 40% | 时间差越大分越高 |
| 催化剂 | 20% | 是否有明确的近期触发因素 |
| 估值/流动性 | 10% | 是否便宜、能否进出 |

## 六、核心持仓

### 1. $AXTI — AXT Inc（InP 衬底）

- **定位**：全球 InP 衬底产能占比 25%-35%，霍尔木兹海峡级别的卡位
- **逻辑**：AI 光通信依赖 InP 衬底，没有它底层材料链断掉
- **时间差验证**：扩产 18-24 个月，光模块翻倍增长 → 完美匹配
- **买入时**：市值约 $2 亿，几乎无机构覆盖
- **路径**：$12 → $70+（最高浮盈 1000%+），目标价 $150
- **状态**：已大幅兑现利润

### 2. $SIVE — Sivers Semiconductors（CPO 激光光源）

- **定位**：CPO 架构外置连续波激光光源，最稀缺上游资产之一
- **买入时**：市值约 $1.5 亿
- **涨幅**：20x+
- **催化剂**：$AVGO 或 $MRVL 可能以 $2-3 亿收购，锁定 CPO 激光供应链
- **状态**：当前最坚定持仓

### 3. $AAOI — Applied Optoelectronics（光收发器）

- **定位**：光收发器全链条玩家（激光器设计 → 组装 → 销售）
- **买入时**：市值约 $6.6 亿，认为被严重低估
- **涨幅**：7x+
- **逻辑**：AI 光模块需求爆发
- **状态**：监控中

### 4. $RPI — Raspberry Pi（边缘计算基座）

- **定位**：大多数认为是廉价教育单板电脑 → 实际是 AI 初创公司的物理隔离部署基座
- **战绩**：提前 2 个月预测全年营收增速 58%（市场一致预期 14%），财报后单日 +44%，次日 +27%

### 5. $XFAB — X-FAB（SiC 代工 + 硅光子）**最新建仓**

- **定位**：欧美唯一高量产 SiC 代工厂，同时具备硅光子代工能力
- **市值**：约 $12.8 亿
- **市净率**：1.29
- **催化剂**：EU CHIPS Act 2 预计近期落地；英伟达和诺基亚硅光子评估进行中
- **标签**：小市值、无覆盖、物理瓶颈、等待机构轮动验证

## 八、整体画像

- **选股风格**：小市值（$1-15 亿）、无机构覆盖、冷门赛道
- **底层主线**：AI 数据中心光互连 + CPO 架构
- **研究方法**：材料学论文 + 专利分析 + 供应链追踪 + 出口管制
- **验证方法**：AI 对抗性压力测试（找漏洞/替代威胁/估值偏差）
- **首要筛选**：融资结构与流通盘动态（ATM/债务/SBC）
- **核心时间差**：扩产周期（2年+）vs 需求增速（翻倍级）
- **发布形式**：X 长线程（深度研究帖）
- **仓位特征**：25 只公开建仓，涨幅 1x-10x
- **最大风险**：高度集中（底层逻辑一致），若 CPO 路线变或推进慢，组合同时受影响
- **流动性风险**：小公司流动性有限，大资金建仓/退出困难
